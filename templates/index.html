What we've done in Y2 CS so far

We're almost halfway through MEET Y2 CS! You will be starting work on the projects soon.

We started off learning about HTML and CSS so that we could make websites. This is called the "frontend", and it's the part of the webapp that your users see when they log on.

Then, we did a review of Python and learned about relational databases. We connected Python with databases using something called SQLAlchemy. Databases store all the information for your webapp.

After talking about databases, we learned how to build a webserver using Flask. Web servers take whatever you type in to the browser (for example, like http://tinyurl.com/meety2cs), and use Python functions to show you the right HTML in your browser.

Now, we're learning JavaScript. We can make dynamic pages that change when we click on things or type data in boxes.

Technologies that you know:

  • HTML
  • CSS
  • Python
  • SQLAlchemy
  • Flask
  • JavaScript
  • jQuery
